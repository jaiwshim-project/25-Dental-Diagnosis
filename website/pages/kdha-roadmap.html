대한치과위생사협회
조직 작동력 진단 및 워크숍

전체 추진 로드맵 (확정안)

대한치과위생사협회 조직 작동력 진단은 총 5단계로 구성됩니다. 각 단계는 인터뷰, 설문, 분석, 보고, 워크숍으로 이어지며 조직의 실제 작동 구조를 정확히 파악하고 지속 가능한 운영 기준을 수립합니다.

핵심 목표 — 진단의 방향성과 정치적 중립성을 확보하고, 조직이 스스로 작동하는 기준을 만듭니다.

1단계

계약 체결 + 1차 대면 인터뷰 (핵심 리더)

목적

진단의 방향성과 정치적 중립성 확보
조직 작동 구조에 대한 1차 가설 설정

① 계약 체결

  • 수행 범위 확정
  • 일정 확정
  • 비밀보장 조항 명문화
  • 자료 제공 범위 합의

② 계약 직후 1차 대면 인터뷰

대상
회장 1명 (필수) 부회장 2~3명
목적
  • 조직 운영 철학 확인
  • 현재 가장 큰 고민 확인
  • 의사결정 구조에 대한 리더 인식 파악
  • 민감 이슈 사전 파악
이 단계는 "방향 정렬 단계"입니다.
2단계

온라인 서면 인터뷰 (전원 대상)

목적

조직 전체 인식 구조 파악

대상

회장단 전원 임원 전원 사무처 직원 전원
방식

온라인 설문 기반 서면 인터뷰

내용 구성

  • 조직 기준 명확성
  • 의사결정 구조 인식
  • 역할·책임 경계 인식
  • 갈등 반복 영역
  • 실행력 체감도
  • 신뢰·공정성 인식
이 단계는 "인식 지도 작성 단계"입니다.
3단계

2차 대면 심층 인터뷰 (선별 진행)

목적

구조적 이슈 심층 분석

선정 기준

  • 서면 응답에서 인식 차이가 큰 인원
  • 의사결정 핵심 참여자
  • 갈등 발생 영역 관련자

예상 인원

회장 (추가 심층 필요 시) 부회장 1~2명 이사 2~3명 사무처 직원 3~5명
방식

하루 집중 인터뷰 (대면 또는 화상) · 각 60~90분

핵심 질문

  • 실제 의사결정은 어떻게 이루어지는가?
  • 공식 구조와 실제 구조의 차이는 무엇인가?
  • 병목은 어디에서 발생하는가?
  • 권한은 명확한가?
  • 갈등은 왜 반복되는가?
이 단계는 "구조 실측 단계"입니다.
4단계

온라인 조직진단 (전원 대상 정량 진단)

목적

정량 데이터 확보 및 구조 검증

대상 및 구성

임원 및 사무처 전원
7대 진단 영역 기반 정량 설문
  • 방향성과 기준
  • 의사결정 체계
  • 역할·책임 명확성
  • 집행부–사무국 작동 구조
  • 세대 간 기준 차이
  • 갈등 처리 체계
  • 실행력 및 신뢰도

진단 결과 산출

  • 영역별 점수
  • 직군별 차이 분석
  • 인식 격차 분석
  • 리스크 지점 도출
이 단계는 "데이터 확정 단계"입니다.
5단계

분석 → 보고 → 리뷰 → 워크숍

① 통합 분석

  • 서면 인터뷰 결과
  • 심층 인터뷰 결과
  • 조직진단 데이터
  • 의사결정 사례 분석
  • 업무 흐름 맵핑
  • 통합 구조 분석

② 진단 보고서 제출

  • 조직 작동력 종합 점수
  • 핵심 병목 구조
  • 반복 갈등 원인
  • 즉시 개선 영역
  • 구조 개선 권고안
  • 3개월 실행 과제 제안

③ 회장단 리뷰 세션

  • 보고서 설명
  • 민감 영역 조율
  • 우선순위 확정
  • 워크숍 방향 확정

④ 직원 워크숍 진행

목적
  • 기준 정렬
  • 역할 경계 재정의
  • 의사결정 원칙 합의
  • 실행 과제 확정
